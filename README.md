# breast-cancer-ml-diagnosis
Machine learning project for breast cancer diagnosis using the Wisconsin dataset using classification and statistical methods
# Breast Cancer ML Diagnosis

This project investigates how machine learning models can be used to predict breast cancer diagnosis using the Breast Cancer Wisconsin dataset.

## Project Goal
The main goal is to evaluate how reliable machine learning models remain when fewer tumour measurement features are available.

## Methods
- Exploratory Data Analysis
- Statistical Analysis
- PCA
- Logistic Regression
- Random Forest
- Support Vector Machine
- Feature Reduction Experiments

## Dataset
Breast Cancer Wisconsin (Original) dataset from UCI Machine Learning Repository.

## Tools
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
